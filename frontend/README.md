# Run Empire - Frontend

This is the Next.js frontend client for the Run Empire territory control game.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure local environment variables in `.env`:
   ```env
   NEXT_PUBLIC_BACKEND_URL=http://localhost:3001
   ```
3. Start the Next.js development server:
   ```bash
   npm run dev
   ```

For full environment configuration and backend instructions, refer to the root [README.md](../README.md).
